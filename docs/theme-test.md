# Praxis Theme — Test Note

Use this note to visually verify calloutstyles, custom checkbox states, and progress bars after changes to `theme.css` and `publish.css`.

Test in: **Dark mode**, **Light mode**, **True Black** (`praxis-contrast-black`).
Test callout style variants: **Soft**, **Outlined**, **Minimal** (Style Settings).

---

## Callouts — 7 semantic groups

> [!note] Purple — note
> `[!note]` intellectual / reference

> [!abstract] Purple — abstract
> Also: `[!summary]`, `[!tldr]`

> [!info] Blue — info
> `[!info]` informational / neutral

> [!todo] Blue — todo
> `[!todo]` actionable / neutral. Also: `[!example]`

> [!example] Blue — example
> `[!example]` neutral demonstration

> [!tip] Teal — tip
> `[!tip]` helpful. Also: `[!hint]`

> [!success] Teal — success
> `[!success]` completed. Also: `[!check]`, `[!done]`

> [!question] Yellow — question
> `[!question]` uncertain / inquiry. Also: `[!help]`, `[!faq]`

> [!warning] Orange — warning
> `[!warning]` caution. Also: `[!caution]`, `[!attention]`

> [!important] Orange — important
> `[!important]` urgent attention (orange, not teal)

> [!failure] Red — failure
> `[!failure]` negative. Also: `[!fail]`, `[!missing]`

> [!danger] Red — danger
> `[!danger]` critical. Also: `[!error]`

> [!bug] Red — bug
> `[!bug]` technical issue

> [!quote] Brown — quote
> `[!quote]` attribution. Also: `[!cite]`

## Custom checkbox states

### Status / workflow

- [ ] incomplete (empty)
- [x] complete (faded, no line-through)
- [/] in-progress (half-fill)
- [-] canceled (strikethrough, muted)
- [>] forwarded / delegated
- [<] scheduled / rescheduled

### Signals

- [?] question / uncertain
- [!] urgent / important
- [*] starred / highlight
- ["] quote reference

### Actionable

- [t] timed / waiting
- [a] alarm / deadline
- [l] location
- [P] phone call
- [R] review / due
- [+] added / created

### Categorical

- [i] information
- [S] money / savings
- [I] idea / insight
- [B] brain / mental model
- [b] bookmark
- [n] note / annotation
- [k] key insight / access
- [w] win / achievement

### Sentiment

- [p] pros / positive
- [c] cons / negative
- [u] trending up
- [d] trending down
- [L] love / value
- [f] fire / hot topic

## Progress bars [0]–[9]

- [0] 0% — empty bar
- [1] 10%
- [2] 20%
- [3] 30%
- [4] 40%
- [5] 50% — half
- [6] 60%
- [7] 70%
- [8] 80%
- [9] 90% — nearly full (intentional: no 100% state)
